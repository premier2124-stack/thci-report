# THCI 데일리 리포트

한국증시 데일리 마켓 리뷰 모음 (개인 보관·열람용). `reports/`의 마크다운을 `index.html` 뷰어가 GitHub Pages에서 렌더해 보여준다.

- 🌐 사이트: `https://<user>.github.io/<repo>/` (GitHub Pages)
- 📄 새 리포트 추가: `reports/`에 `YYYY-MM-DD_*.md` 넣고 push → 자동 목록 반영
- 🔄 동기화: `update.ps1` 실행 (OneDrive 리포트 폴더 → reports/ 복사 후 commit·push)
- 🔒 검색 차단: index.html에 `noindex` (링크 아는 사람만 사실상 열람). 공개 repo이므로 민감정보(계좌·키)는 절대 넣지 말 것.

> 본 자료는 공개 시장 데이터·언론 보도를 정리한 개인 시황 메모이며, 특정 종목 매매를 권유하지 않습니다..
